# England & Wales microcensus analysis of nurses’ commuting patterns

## Summary
This component contains public materials for the completed England & Wales microcensus analysis of nurses’ commuting patterns. Materials include study documentation and links to the archived analysis code.

## Study status
Status: completed analysis  
Manuscript status: submitted to *International Nursing Review* and currently under peer review  
Preregistration status: none for this England & Wales analysis, as the study was completed before the current prospective open-science workflow was established

## Data access
The underlying England & Wales microcensus data used for this analysis are restricted-access and cannot be shared publicly through OSF or GitHub. This component therefore provides documentation and code-access metadata only.

## Reproducibility note
The code is shared to improve transparency and reproducibility. Full rerunning of the analysis requires access to the restricted source data and may require local modification of file paths, package versions, and environment settings.

## Related project structure
A separate OSF component is being used for the Scotland analysis, which will follow a prospective open-science workflow including preregistration.
